> Difficulty : *Medium*

### Statement
Given an integer $N$, find its integer square root.

$int\\_sqrt(N) = \lfloor \sqrt N \rfloor$

- $0 \le N \le 10^5$

### Example

- int_sqrt(1) = 1
- int_sqrt(16) = 4
- int_sqrt(42) = 6

### Complexity
- Time : $O(\log N)$

### Follow-up
Given a real number $X$, find its square root with a precision of $10^{-3}$.

- $0 \le X \le 10^5$
